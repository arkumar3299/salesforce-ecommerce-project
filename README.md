Salesforce E-Commerce Customer Support System

## About the Project
This project is built using Salesforce to manage customer orders and support cases in an E-commerce system.

## Features
- Customer and Order Management
- Automatic Case creation when order is cancelled
- Email notifications using Flows
- Dynamic UI using Lightning Web Components (LWC)

## Technologies Used
- Salesforce CRM
- Apex (Triggers)
- Lightning Web Components (LWC)
- Flows
- SOQL

## My Work
- Created custom objects like Customer, Order, and Order Item
- Implemented automation using Flows
- Developed Apex Trigger for business logic
- Built LWC for displaying order details
- Created Reports and Dashboards

## Output
- Reduced manual work using automation
- Improved tracking of orders and support cases
